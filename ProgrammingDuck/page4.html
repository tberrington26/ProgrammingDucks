Text and Font

CSS allows for the user to change the

font
of words, and change the
sizes
of the words.

In order to change the font of words, the code font-family: Arial can be used. This code snippet allows for the user to place back-up fonts in case one of them is unavailable for the web browser. There are a large amount of font types that can be used, if you would like more ideas, you can google them.

font-family: Impact, Haettenschweiler, 'Arial Narrow Bold';

Head down into the bottom window and add a window where you can insert this code. Feel free to change the font of any of the words on this webpage.

After you have sufficiently messed with the different font types, look for the sizing of the fonts.

The code to change the font size of different elements is font-size: 16px. You can utilize different pixels to get the look you want on the webpage.

You can head into the bottom window and add in the code to mess with the sizing.

font-size: 16px;

There is another bit of code that allows you to make the text italic and bold. If you put font-style:italic or bold you can make the font look as such.

font-style: italic;

font-style: bold;

Once you find the link, you can move on!
