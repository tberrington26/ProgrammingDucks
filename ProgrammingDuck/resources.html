Name Link Explanation
Code Academy Code Academy Code Academy offers a wide variety of coding lessons. If you ever have any desire to learn any language, or need help, Code Academy most likely has lessons on that topic.
Code.org Code.org Code.org offers quite a few lessons based on javascript, each lesson is different from another, and it will help you gain the basics of understanding what coding is.

W3 Schools W3 Schools W3 Schools is a resource that answers questions and has small lessons on different programming languages. The link brings you to HTML practice lessons instead of the homepage.
Playcode Playcode.io This is an online browser for html coding. This will allow you to practice coding in HTML/CSS without having to download anything
