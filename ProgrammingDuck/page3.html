Position refers to the location of an element on the webpage. If you scroll down in the bottom window of Google Devs, you will come across a multicolor box that contains the four sections of the box model.

Some pieces of code that can affect the position of an element are the padding, margin, width/height, and border bits of code.

The long box uses the padding bit of code. It functions very similarily to what the margin code does, both in how the code looks, and what it appears to do, however, padding is the space in between the border and the element (which elongates the box), and margin is the space outside of the border.

padding: 10px;

The code will look like padding:10px for every side of the element or if you want different sizes for each side, then you can use padding: 1px 2px 3px 4px or any variation of the sizes.

padding: 1px 2px 3px 4px;

Select an element and change the padding elements, either through the dashed lines in the box model or with the code snippets!

The middle box uses margin code to affect the space around it.

margin: 10px;

You can also change the margin of an element by selecting the element, clicking the plus sign, and then typing in the box, margin: 10px. This will affect every margin for the element. If you want each margin to be different, you can use margin: top right bottom left. Just put in the desired sizes and move on.
